# META-FOUNDRY Architecture — Level L3 (Processes)
Документ: MF-ARCH-L3
Версия: 1.0.0


## Назначение
L3 описывает процессы, которые связывают компоненты L2.


## Основные процессы
- Git Flow (feature → develop → release → main)
- MSDLC Flow (MF-0 → MF-6)
- AutoDoc Flow (watch → generate → update → publish)
- Security Flow (scan → verify → enforce)
- Build Flow (dependencies → build → sign → SBOM)
- Release Flow (review → gates → publish)
