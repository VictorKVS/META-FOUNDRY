## Quantum Data Sphere v3 — META-FOUNDRY Banner

Футуристический фирменный баннер в стиле Palantir Foundry.  
Используется на главной странице и архитектурных разделах портала META-FOUNDRY.

![Quantum Data Sphere v3](../assets/banners/quantum-data-sphere-v3.png)

**Идея:**  
Сфера данных с многоуровневыми орбитами отражает структуру META-FOUNDRY:  
Core → Intelligence → Governance → Lifecycle → Human+ → Labs.

**Применение:**  
- Главная страница портала  
- Архитектура и концептуальные схемы  
- Презентации и бренд-бук  
- Маркетинговые материалы MindForge/META-FOUNDRY
