Git Governance & Secure Engineering Standard
Документ: MF-ST-01.01
Версия: 1.0.0
Статус: ACTUAL / APPROVED


Этот документ определяет корпоративный Git-процесс META-FOUNDRY, включая:
- стратегию ветвления
- PR-процессы
- стандарты коммитов
- CI/CD практики
- DevSecOps контроль
- Golden Security Rules
- MSDLC → Git mapping


## Состав стандарта
- Branching Strategy
- Commit & PR Standards
- CI/CD Integration
- Security Requirements
- AutoDoc Integration
- Git Maturity Model (см. приложение C)
