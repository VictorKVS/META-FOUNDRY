# MSDLC Standard (MindForge Software Development LifeCycle)
Документ: MF-ST-02.01
Версия: 1.0.0


MSDLC — корпоративный жизненный цикл разработки МайндФордж, включающий стадии:
1. MF-0 — Initiation
2. MF-1 — Analysis
3. MF-2 — Architecture
4. MF-3 — Build
5. MF-4 — Verification
6. MF-5 — Release
7. MF-6 — Knowledge


## Требования MSDLC
Каждая фаза имеет:
- цели
- входы
- выходы
- артефакты
- архитектурные правила
- точки контроля безопасности
