# META-FOUNDRY — Style Guide

## Headings
- H1 — Название раздела  
- H2 — Подраздел  
- H3 — Логические блоки (архитектура, паттерны)

## Typography
- Используем **жирный** для ключевых понятий  
- *Курсив* — для определения или философских элементов  
- Кодовые блоки — ```  
- Таблицы — 3 колонки минимум  

## Voice & Tone
- Строго  
- Инженерно  
- Корпоративно  
- Без воды  
- Фокус на архитектуре, масштабируемости, безопасности  

## Structural Rules
- Каждый README начинается с 3 строк описания  
- Каждая папка имеет таблицу навигации  
- Все документы = Markdown
